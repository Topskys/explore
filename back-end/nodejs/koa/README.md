# koa

## 脚手架
```bash
npm install -g koa-generator
```

## 使用koa2创建koa项目
```bash
koa2 demo
```

## 运行
```bash
cd demo
npm install
npm run dev
# or
npm start
```
访问地址：http://localhost:3000/
